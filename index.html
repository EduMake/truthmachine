Using TruthMachine

Keyboard Shortcuts

You can use the following keys to type operators:

  • ~ or !
  • * or .
  • v or |
  • >
  • =

Variables

You can use any alphabetic character (you don't have to use 'p' and 'q'!) as variables

Parentheses

You can use any combination of (rounded), {curly} or [square] brackets to designate precendence.
